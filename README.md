# SocialApp
A web application with MEAN Stack (MongoB, Express, Angular, NodeJS)

<hr>

## Technology Stack
- Front-end - Angular 6, Materialize CSS
- Back-end -  Express and NodeJs
- Databases - MongoDB

<hr>

## Features

- RESTful API Design
- JSON Web Tokens for Authentication
- Authorization 
- Image Upload
- Chat functionality

<hr>
